# Feature: Structured Export to resume.lol Format

## What
An exporter that outputs the adapted resume in a structured Markdown format. This format is preconfigured to match the layout and schemas required for a direct copy-paste or upload into the resume.lol visual resume editor.

## Why
Formatting a resume is tedious. By exporting directly to a compatible, beautiful visual template platform like resume.lol, users get a high-quality visual resume instantly without design effort.

## How
- Map internal Go resume models to resume.lol Markdown layouts.
- Generate clean output files with customizable destination paths.

## When
Implemented fourth, providing the final output capability for the application.
